# Rust 所有权

Rust 的核心特性：**不需要 GC 也能保证内存安全**。

## 三条规则

1. 每个值在 Rust 中都有一个**所有者**（owner）
2. 一次只能有一个所有者
3. 当所有者离开作用域（scope），这个值被丢弃

## 示例

```rust
fn main() {
    let s = String::from("hello"); // s 是这个 String 的所有者
    takes_ownership(s);            // s 的所有权移入函数
    // println!("{}", s);          // 错误！s 已经无效
    
    let x = 5;
    makes_copy(x);                 // x 是 i32，复制了一份
    println!("{}", x);             // 仍然有效
}
```

## 闪卡

所有权三规则是什么？ #card
- 每个值有唯一所有者；离开作用域值被丢弃；同一时刻仅一个所有者。

为什么 Rust 不需要 GC？ #card
- 编译期通过所有权系统在编译期追踪每个值的使用，超出作用域自动 drop，零运行时开销。
