# Rust 生命周期

生命周期是 Rust 用来确保**所有引用都有效**的机制。是 [[Rust 借用]] 系统的一部分。

## 为什么需要？

```rust
fn longest(x: &str, y: &str) -> &str {  // 缺生命周期标注
    if x.len() > y.len() { x } else { y }
}
```

编译器无法判断返回的引用指向 x 还是 y，需要程序员标注。

## 标注方式

```rust
fn longest<'a>(x: &'a str, y: &'a str) -> &'a str {
    if x.len() > y.len() { x } else { y }
}
```

`'a` 表示返回的引用至少和参数活得一样久。
