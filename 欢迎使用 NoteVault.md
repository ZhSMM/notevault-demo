# 欢迎使用 NoteVault

这是一个**本地优先**的笔记应用，你的所有笔记都存在你指定的目录里（我们叫它"vault"）。

## 快速上手

- **左下角加号** → 新建笔记
- **Ctrl+P** → 全局搜索
- **Ctrl+S** → 保存当前笔记
- 笔记里写 `[[另一篇笔记]]` → 创建链接
- 笔记里写 `#标签` → 自动分类

## 试试双向链接

打开 [[Rust 所有权]]，里面有指向 [[Rust 借用]] 和 [[Rust 生命周期]] 的链接。

## 试试搜索

按 **Ctrl+P**，试试搜 "所有权" 或 "rust"。

## 数据存在哪？

你打开 vault 时选的那个目录。里面就是普通的 `.md` 文件，你可以用任何编辑器（VSCode、Typora 等）打开。
应用在 `.notevault/` 子目录里建了一个 SQLite 索引，**不修改你的笔记文件**。
