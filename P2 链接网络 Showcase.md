# P2 链接网络 Showcase

> 试试在右边预览里：
> - **hover 任意 wikilink**（`[[xxx]]`）—— 0.5 秒后弹出目标笔记的摘要
> - **hover 任意段落** —— 左边出现 `#` 锚点，点击它会把 `((blk_xxxx))` 复制到剪贴板
> - **底部"反向链接"面板** —— 显示所有引用本篇的笔记 + 上下文

## 反向链接

打开这篇笔记，看底部 ——
- 如果别处有写 `[[P2 链接网络 Showcase]]` 或 `[[渲染能力 Showcase]]`，它们会自动出现在这里
- 每条都带**上下文**（哪一段提到了我）

## 块引用

我在底下偷偷放了一个 `((blk_xxxxxxx))` 引用，hover 它会高亮，点击会跳到那个块（带闪烁效果）。

> 试着自己写一篇笔记，里面写 `[[P2 链接网络 Showcase]]`，保存后这篇笔记底部就会出现你的笔记。

## 跨笔记跳转

点 `[[Rust 所有权]]` 试试 —— 会跳到 Rust 所有权那篇笔记里。

带章节的：`[[Rust 所有权#关联概念]]` —— 会跳到那篇笔记的"关联概念"小节。

## 悬空链接

如果 wikilink 指向不存在的笔记，左下角面板会列出所有悬空链接，方便你补上或删除。

## 已知限制

- 块 ID 基于内容 hash + 顺序生成，**改了内容 ID 会变**（引用就悬空了）
- `((block-id))` 引用需要先知道 block ID（hover 段落左侧 `#` 复制）
- transclusion `![[note]]` 的实际嵌入渲染留到下一轮

## 一段示例

```rust
// 普通代码块还是会高亮
fn main() {
    println!("hello P2");
}
```
